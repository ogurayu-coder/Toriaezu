# 🐾 わんこノート

犬の体調・お世話を記録するシンプルなWebアプリです。ごはん・散歩・うんち・お薬・気になったことを、その場でサッと記録できます。特にシニア犬や持病のある子の「あのとき何があったか」を後から見返せるようにする目的で作っています。

アプリはブラウザだけで動作し、サーバーには何も送信しません。記録はすべて**そのブラウザ・その端末の中(localStorage)**に保存されます。

---

## ファイル構成

| ファイル | 用途 | URL |
|---|---|---|
| `index.html` | 本番。実際の記録用 | https://ogurayu-coder.github.io/Toriaezu |
| `demo.html` | お試し用。新機能やデザインの確認、友達への案内用 | https://ogurayu-coder.github.io/Toriaezu/demo.html |

`index.html`と`demo.html`は保存先のキー名が別々になっているため、**同じ端末で両方開いてもデータが混ざることはありません**。デザインや「犬を管理」機能を試したいときは、必ず`demo.html`の方を使ってください。

---

## データの保存について(重要)

- 記録は**そのブラウザ・その端末だけ**に保存されます(クラウド同期はしていません)。
- 同じURLでも、iPhoneとiPadのように端末が違えば別々のデータになります。
- **プライベートブラウズモード(シークレットモード)で開くと、タブを閉じた瞬間にデータが消えます。** 普段使いは必ず通常モードのSafariで。
- Safariの「履歴とWebサイトデータを消去」を行うと、記録も消えます。
- ホーム画面にアイコンを追加しておくと、毎回同じ場所からアクセスできて安心です(Safariの共有ボタン →「ホーム画面に追加」)。

## バックアップ

アプリ内の「バックアップを表示」から、記録+犬のリストをまとめてテキストとして書き出せます。長押しでコピーし、メモアプリなどに保存しておくと安心です。「バックアップを復元」に貼り付ければ元に戻せます。

**犬を追加・削除する前は、念のためバックアップを取っておくことをおすすめします。**

---

## 友達に使ってもらう場合

このURL(本番またはデモ版)をそのまま教えるだけでOKです。GitHubアカウントは不要です。

- 友達がそのURLを初めて開くと、「わんこノートへようこそ」という**セットアップ画面**が表示され、飼っている犬の名前・耳の形・模様・毛色を登録できます。
- Mo(オーナー)のiPhoneではすでに犬のデータがあるため、このセットアップ画面は出ず、いつも通り使えます。
- 一度セットアップした後は、アプリ内の「🐾 犬を管理」から、いつでも犬の追加・編集・削除ができます(保存を押すまでは確定しません)。

---

## 主な機能

- ごはん・散歩・うんち・お薬・気になったことの記録(時間・メモ付き)
- カレンダー表示(気になったことがあった日が一目でわかる)
- 犬の名前・日付・メモでの検索
- 過去の日付にさかのぼって記録を追加(カレンダーの日付をタップ)
- バックアップの書き出し・復元
- 「🐾 犬を管理」での犬の追加・編集・削除
